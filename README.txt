Stock Price Analyzer Program
=============================

Language used: Java 
Source files:
	(1) StockPriceAnalyzer.java
	(2) SecurityInfo.java
	(3) DayTransaction.java
	
Instructions for Running
------------------------

(a) Place the three source files in a folder.
(b) Compile the source code with the command: "javac StockPriceAnalyzer.java"
(c) Run the program. The command is the following.
java StockPriceAnalyzer --max-daily-profit --busy-day --biggest-loser
